## `firelayer init [NAME]`

create a new project

```
USAGE
  $ firelayer init [NAME]

OPTIONS
  -h, --help  show CLI help
  -s, --skip  skip install deps

EXAMPLE
  $ firelayer init new-firelayer-app
```

_See code: [dist/commands/init.ts](https://github.com/firelayer/firelayer/blob/v1.0.0-alpha.4/dist/commands/init.ts)_
