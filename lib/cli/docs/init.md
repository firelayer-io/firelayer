## `firelayer init [NAME]`

create a new project

```
USAGE
  $ firelayer init [NAME]

OPTIONS
  -h, --help               show CLI help
  -s, --skip               skip install deps
  -t, --template=template  [default: starter] template to install

EXAMPLE
  $ firelayer init new-firelayer-app
```
